# Psiu API

## Requisitos

### Aluno
- [] Cadastrar um aluno
 - [] Não deve ser possível cadastrar RA já cadastrado

- [] Editar aluno
 - [] Deve ser possível alterar o nome e data de nascimento do aluno

- [] Atualizar senha
 - [] Não deve ser possível colocar uma senha fraca

- [] Deletar aluno
 - [] Deve utilizar soft delete
 
- [] Buscar alunos

### Post

- [] Criar post
- [] Editar post
- [] Deletar post
- [] Buscar posts
- [] Buscar posts por aluno

### ComentarioPostagem

- [] Criar comentário
- [] Editar comentário
- [] Deletar comentário

### Reacaodepoimento

- [] Criar reação
- [] Deletar reação
